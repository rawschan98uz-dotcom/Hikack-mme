# Project Brief — Hijack-mme

Local ModMe-like CRM clone for education centers (Uzbekistan market).

**Goal:** Functional parity with ModMe web app — navigation, CRUD, and cross-module links — not pixel-perfect UI.

**Stack:** Django REST + JWT backend, Vue 3 + Tailwind frontend, SQLite locally.

**Tenant:** Single demo company `ravvatech` (RavvaTech).

**Implementation plan:** Sequential phases 0–15; see `docs/00-phase0-audit.md`.
